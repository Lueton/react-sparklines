## Properties

See [SparklinesLine API](/docs/api/sparklines/sparklines-line) for shared properties (`data`, `margin`, `width`, `height`, `clip`, `min`, `max`, `responsive`, `startAtZero`, `zeroBaseline`, `preserveAspectRatio`, events, etc.).

SparklinesBand-specific properties include `curved`, `dots`, `activeDot`, `fill`, `fillOpacity`, `stroke`, `strokeWidth`, and `style` which behave identically to their SparklinesLine counterparts but applied to the band shape.
